Car Rental gRPC API (Proto)
//This file defines the Protocol Buffers schema for a car rental service.
//It specifies the data models messages, enums, and the gRPC service methods that clients and servers use to communicate.

Features:
//Car management: add, update, remove, search, list available cars
//User management: create single/multiple users (via client streaming)
//Cart management: add cars with rental dates
//Reservation system: place reservations and return details
//Supports both client streaming (for users) and server streaming (for listing cars)
